营养成分分析

Analysis of Nutrients

氨基酸：

北极野生海参含有多种氨基酸，并可促进细胞的再生和机体损伤后的修复，还可提高淋巴细胞的免疫活性，增强人体免疫力，消除疲劳，延缓衰老，具有改善脑，性腺神经功能传导作用，减缓性腺衰老，提高勃起能力，增强生血，养血，补血作用。

海参素：

北极野生海参的海参素含量丰富，其作用为：抗肿瘤，抗菌，增强免疫力，调节造血功能，促进性功能，抗疲劳，治疗炎症，护肝保血管，成骨，抗癌，有效保护心脏。对肝炎，结核病，糖尿病，心脑血管病有显著的治疗作用。

海参粘多糖：

海参粘多糖的主要作用：抗菌，防癌，增强免疫力，抗血栓，降血糖，延缓衰老，抗病毒，抗辐射，可防止骨质疏松症，对关节炎有治疗作用。

脂肪酸、蛋白质、维生素、矿物营养成分：

北极野生海参含有人体必须的脂肪酸，花生四烯酸和EPA，有益智健脑，养血润燥，修补组织的功效。海参中主要含有蛋白质和维生素A，B，D，E，及大量的矿物营养成分，如钙，镁，锰，硅等，为天然补钙食品。
